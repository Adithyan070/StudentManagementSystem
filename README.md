Student Management System (Java + JDBC + MySQL)

This is a simple console-based Student Management System built using Java, JDBC, MySQL, and Maven.
It performs basic CRUD operations such as adding, viewing, updating, and deleting student records.

1. Features

    -Add new student

    -View all students

    -Search student by ID

    -Update student details

    -Delete student

    -Uses DAO and Service layers

    -Uses PreparedStatement for database security

2. Technologies Used:

    -Java 17 or above

    -JDBC

    -MySQL

    -Maven

3.Concepts Learned:

    -Java OOP

    -JDBC connectivity

    -CRUD operations

    -DAO & Service Layer

    -Exception handling

    -Maven build lifecycle
